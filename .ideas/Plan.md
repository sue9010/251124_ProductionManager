📋 COX Production Manager v2.0 - Grand Master Plan

🎨 Phase 0: 시스템 기초 & 디자인 (System Foundation)

목표: 프로그램의 뼈대를 튼튼히 하고, 전반적인 디자인 시스템을 구축합니다.

[ ] 0.1. 스타일 모듈화 (styles.py)

[ ] 색상(Primary, Danger, Success 등) 및 폰트 상수 정의.

[ ] 버튼, 라벨, 입력창 등 공통 위젯 스타일 딕셔너리 구현.

[ ] 0.2. 팝업 구조 표준화 (base_popup.py)

[ ] StandardPopup 클래스 구현 (Header-Info-List-Footer 4단 레이아웃).

[ ] 기존 팝업들(Schedule, Complete, View)을 상속 구조로 리팩토링.

[ ] 0.3. 토스트 알림 시스템 (Idea #24)

[ ] ToastManager 클래스 구현 (화면 하단 Fade-in/out 알림).

[ ] 기존 messagebox를 비침해적(Non-intrusive) 알림으로 대체.

[ ] 0.4. 워크스페이스 상태 기억 (Idea #42)

[ ] 앱 종료 시 윈도우 크기, 위치, 필터 상태를 config.json에 저장.

[ ] 앱 실행 시 저장된 상태 복원.

[ ] 0.5. 스플래시 스크린 (Idea #99)

[ ] 앱 실행 시 로고와 로딩 상태를 보여주는 인트로 윈도우 구현.

🛠️ Phase 1: 데이터 로직 강화 (Data Logic)

목표: 데이터의 무결성을 보장하고 관리 로직을 고도화합니다.

[ ] 1.1. 용어 및 로직 리팩토링

[ ] "Hold" → "작업 중지"로 용어 일괄 변경.

[ ] 구버전 데이터 로드 시 자동 마이그레이션 로직 추가.

[ ] 1.2. 데이터 타임머신 (Idea #3)

[ ] 저장 시마다 .ProductionManager/backups/ 폴더에 타임스탬프 파일 자동 백업.

[ ] 1.3. 입력 데이터 유효성 검사 (Idea #14)

[ ] 날짜 형식(YYYY-MM-DD), 필수값 누락 방지 로직 강화.

[ ] 1.4. 엑셀 변경 감지 (Idea #15)

[ ] 외부에서 파일이 수정되었을 경우 감지하여 새로고침 알림 표시.

[ ] 1.5. 중요 항목 상단 고정 (Idea #23)

[ ] 중요(Pin) 상태 관리 로직 추가 및 정렬 시 최우선 순위 배정.

[ ] 1.6. 자연스러운 정렬 (Idea #97)

[ ] natsort 알고리즘 적용 (A1, A2, A10 순서 정렬).

[ ] 1.7. 대문자 자동 변환 (Idea #78)

[ ] 모델명/시리얼 입력 시 소문자를 쳐도 자동으로 대문자로 변환.

🏗️ Phase 2: 메인 UI 혁신 (Main UI Renovation)

목표: 사용자가 가장 많이 보는 메인 화면을 직관적이고 편리하게 바꿉니다.

[ ] 2.1. 레이아웃 재구조화

[ ] Header: 로고 및 시스템 아이콘 그룹화.

[ ] Toolbar: 검색창 확장 및 필터 버튼(Segmented Control) 그룹화.

[ ] 2.2. 시각적 편의 기능 (Visual QoL)

[ ] 헤더 고정 (Idea #60): 스크롤 시에도 컬럼명이 상단에 유지되도록 구조 개선.

[ ] 상태별 행 강조 (Idea #55): '작업 중지'(적색), '완료'(회색) 등 배경색 차별화.

[ ] 신규 등록 'N' 뱃지 (Idea #72): 24시간 내 등록된 건에 아이콘 표시.

[ ] 알약 모양 상태 태그 (Idea #63): Status 텍스트 뒤에 둥근 배경 적용.

[ ] 검색어 형광펜 (Idea #73): 검색 결과 텍스트 하이라이팅.

[ ] 빈 상태 디자인 (Idea #91): 데이터 0건 시 안내 아이콘 표시.

[ ] 2.3. 고급 필터링

[ ] 기간 조회 필터 (Idea #43): 시작일~종료일 범위 검색 기능 구현.

⚡ Phase 3: 인터랙션 및 디테일 (Interaction)

목표: 마우스와 키보드 조작을 즐겁게 만듭니다.

[ ] 3.1. 스마트 기능

[ ] 우클릭 컨텍스트 메뉴 (Idea #2): 상태 변경, 복사, 상세 보기 단축 메뉴.

[ ] 스마트 자동 완성 (Idea #6): 업체명/모델명 입력 시 DB 기반 추천.

[ ] 공통 단축키 (Idea #93): ESC(닫기), Enter(저장) 바인딩.

[ ] 스마트 포커스 (Idea #92): 팝업 오픈 시 주 입력창으로 커서 자동 이동.

[ ] 3.2. 리스트 조작 편의

[ ] 말풍선 툴팁 (Idea #56): 마우스 오버 시 잘린 텍스트 전체 표시.

[ ] 컬럼 너비 자동 맞춤 (Idea #64): 헤더 구분선 더블클릭 시 리사이징.

[ ] 현재 행 테두리 강조 (Idea #66): 선택된 행 시인성 강화.

📊 Phase 4: 뷰 모드 및 확장 기능 (Extensions)

목표: 엑셀 리스트를 넘어선 다양한 시각화와 기능을 제공합니다.

[ ] 4.1. 시각화 뷰어

[ ] 칸반 보드 뷰 (Idea #21): Drag & Drop 카드형 UI.

[ ] 간트 차트 뷰 (Idea #41): 일정 막대 그래프 시각화.

[ ] 4.2. 상세 정보 확장

[ ] 커스텀 태그 시스템 (Idea #34): #태그 입력 및 칩(Chip) UI.

[ ] 타임스탬프 댓글 로그 (Idea #44): 채팅형 히스토리 기록.

📤 Phase 5: 출력 및 외부 연동 (Output)

목표: 데이터를 가치 있는 정보로 변환합니다.

[ ] 5.1. 미니 대시보드 (Idea #5)

[ ] 하단바에 핵심 지표(진행률, 중지 건수, 금일 완료) 표시.

[ ] 5.2. 기간별 엑셀 내보내기 (Idea #10)

[ ] 필터링된 현재 리스트를 별도 파일로 저장.

[ ] 5.3. 원클릭 이메일 초안 (Idea #17)

[ ] 완료 건 선택 시 메일 클라이언트 자동 실행 및 내용 작성.
